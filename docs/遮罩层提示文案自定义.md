# 遮罩层提示文案自定义（需求说明）

## 背景
希望支持在设置面板中自定义遮罩层弹窗的提示文案（休息提示），用于替换固定文案，满足个性化提示需求。

## 需求范围
- 覆盖两类遮罩层提示文案：
  - **普通休息/长休息**提示文案
  - **熬夜强制休息**提示文案
- 在 **设置面板** 中提供一个可编辑的多行输入区域。
- 文案持久化到本地（`UserDefaults`），下次启动仍生效。
- 遮罩层展示时按模板渲染，自动注入当前休息类型与分钟数。

## 模板占位符
支持以下占位符（大小写敏感）：
- `{breakType}`：休息类型文本（"休息" / "长休息"）
- `{breakMinutes}`：休息分钟数（例如 18）

默认模板：

```
番茄钟时间到！

{breakType}时间，{breakMinutes}分钟后自动恢复
```

熬夜强制休息默认文案（可在设置中覆盖）：

```
🌙 熬夜时间到了，该休息了！

为了您的健康，请停止工作
强制休息无法取消
```

## 验收标准
- 当用户在设置里编辑模板并保存后：
  - 下一次遮罩层弹出时，提示文案使用新模板渲染。
  - `{breakType}` 与 `{breakMinutes}` 正确替换为当前值。
- 若用户将模板清空（全空白）：
  - 自动回退到默认模板，避免展示空白文案。
