# PomodoroScreen v1.0.2 更新日志

## 🎯 版本信息
- **版本号**: v1.0.2
- **发布日期**: 2026年1月9日
- **类型**: 基于 v1.0.1 的 UI优化 + 问题修复 + 文档补充

## 🚀 新增/变化

### 🚀 新功能（新增）
- **背景随机轮播**：新增随机播放/随机轮播开关，背景素材播放顺序更灵活。
- **支持自定义文案**：支持自定义遮罩层/强制休息提示文案（模板），满足个性化表达需求。

### 🎨 UI 优化（新增）
- **菜单/弹窗暂停图标统一**：状态栏菜单与 popup 控制按钮在“暂停”状态下使用一致的图标风格。
- **专注模式文案统一**：将用户可见的“会议模式”更名为“专注模式”，并优化 tooltip 描述（强调不打断工作、不遮挡屏幕）。
- **专注模式组件居中**：popup 底部“专注模式（开关 + 文案）”整体居中对齐。
- **倒计时显示增强**：popup 中心倒计时文字调大、加粗；分钟不补 0（如 `08:53` → `8:53`）。
- **底部维度信息增强**：四个维度增加右侧百分比数值展示，左右两端对齐，间距更清晰。

### 🐛 问题修复（新增）
- **休眠状态统计未停止**：修复系统休眠/锁屏等场景下统计未正确暂停的问题，避免异常统计数据累积。


## 🔄 兼容性
- macOS 14.0+  
- Apple Silicon + Intel Mac
